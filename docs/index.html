para Hololens 2

Descripción

Se trata de la extesión de realidad aumentada de la aplicación de android LSB UCB.


Esta aplicación tiene la funcionalidad de realizar interpretaciones mediante la captura de voz, realizar evaluaciones al usuario y mantener un registro del progreso de este. Todo esto mientras se brinda una experiencia inmersiva.

Interpretación

La aplicación utiliza un personaje que es el encargado de realizar la interpretación de lo que digamos, es nuestro medio para poder comunicarnos con la aplicacion, el objetivo de la implementación de un personaje es poder generar una experiencia de usuario mucho más cálida e intuitiva para el uso y entendimiento de la misma.

Evaluación

Se realiza la evaluación de un modulo seleccionado. Esta evaluación se realiza de 5 señas generadas de manera randómica de acuerdo al módulo seleccionado.

Perfil

Con esta opción se puede ver la experiencia del usuario y nos abre camino a ver el progreso individual y la tabla de posicionamiento.

Usuarios

La aplicación cuenta con tres diferentes tipos de usuarios para poder seleccionar. Nuevo usuario, dentro de esta opcion se pedira el nombre de usuario para poder comenzar. Usuario existente, esta opción utiliza el código de usuario generado al inicio. Invitado, esta opción permite usar la aplicación de manera temporal, sin guardar ningun dato o información.

Gestos-Iniciar la grabación

Para poder iniciar la grabación de la voz el usuario debera extender su mano y realizar el gesto como se muestra en la imagen.

Gestos-Detener la grabacion

Para poder detener la grabación de la voz el usuario debera extender su mano y realizar el gesto como se muestra en la imagen.
