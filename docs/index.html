para Hololens 2

Descripcion

Se trata de la extesion de realidad aumentada de la aplicacion de android LSB UCB.


Esta aplicacion tiene la funcionalidad de realizar interpretaciones mediante la captura de voz, realizar evaluaciones al usuario y mantener un registro del progreso de este. Todo esto mientras se brinda una experiencia inmersiva.

Interpretacion

Nuestra aplicacion utiliza un personaje que es el encargado de realizar la interpretacion de lo que digamos, es nuestro medio para poder comunicarnos con la aplicacion, el objetivo de la implementacion de un personaje es poder generar una experiencia de usuario mucho mas calida e intuitiva para el uso y entendimiento de la misma.

Evaluacion

Se realiza la evaluacion de un modulo seleccionado. Esta evaluacion se realiza de 5 señas generadas de manera randomica de acuerdo al modulo seleccionado.

Perfil

Con esta opcion se puede ver la experiencia del usuario y nos abre camino a ver el progreso individual y el leaderboard.

Usuarios

La aplicacion cuenta con tres diferentes tipos de usuarios para poder seleccionar. Nuevo usuario, dentro de esta opcion se pedira el nombre de usuario para poder comenzar. Usuario existente, esta opcion utiliza el codigo de usuario generado al inicio. Invitado, esta opcion permite usar la aplicacion de manera temporal, sin guardar ningun dato o informacion.

Gestos-Iniciar la grabacion

Para poder iniciar la grabacion de la voz el usuario debera extender su mano y realizar el gesto como se muestra en la imagen.

Gestos-Detener la grabacion

Para poder detener la grabacion de la voz el usuario debera extender su mano y realizar el gesto como se muestra en la imagen.
